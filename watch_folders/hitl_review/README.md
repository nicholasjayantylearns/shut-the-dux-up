# 📋 HITL Review Folder

Drop files here to trigger **AI analysis and human review workflows**.

## What Happens When You Drop a File:

1. **AI analyzes** the content and structure
2. **Creates a review prompt** with questions for you
3. **Generates a session folder** with all artifacts
4. **Provides response template** for your feedback

## Supported File Types:

- `.json` - Schema files, test data, configurations
- `.md` - Documentation, guides, prompt templates  
- `.yaml/.yml` - Configuration files
- `.txt` - Plain text content
- Any text-based file

## Example Files to Drop:

- New schema definitions
- Updated test data
- Documentation drafts
- Prompt template modifications
- Configuration changes

**Just drop and wait for the AI analysis!** 🚀
