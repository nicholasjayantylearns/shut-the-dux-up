# 💬 HITL Feedback Folder

Drop **completed human response files** here to trigger AI processing of your feedback.

## What to Drop:

- Completed `human_response_template.md` files from review sessions
- Your own feedback/instruction files for AI processing
- Follow-up questions or requests

## What Happens:

1. **AI processes** your feedback and instructions
2. **Creates next steps** and implementation guidance  
3. **Generates action plan** based on your input
4. **Archives** the feedback for reference

## File Naming:

- Use descriptive names: `feedback_schema_updates_20250702.md`
- Include timestamps if multiple feedback rounds
- Reference the original review session if applicable
